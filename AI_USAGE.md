# AI Usage Report

### 1. Em có sử dụng AI không?
Có.

### 2. AI hỗ trợ phần nào?
- Gợi ý khung code HTML/CSS theo Bootstrap 5.
- Khởi tạo mảng dữ liệu mẫu 8 khóa học lập trình.
- Viết hàm JavaScript để tìm kiếm, lọc dữ liệu, mở modal và lưu LocalStorage.
- Fix lỗi CSS bị ẩn/mờ chữ ở phần Hero và Footer.

### 3. Prompt đã sử dụng
- "Thiết kế giao diện web khóa học lập trình bằng Bootstrap 5 phối màu Dark Mode Neon đẹp mắt."
- "Viết code JS lọc dữ liệu theo danh mục và kiểm tra lỗi form đăng ký không dùng alert chung."

### 4. Em đã chỉnh sửa gì sau khi AI sinh code?
- Tách biệt file data.js và main.js theo đúng yêu cầu.
- Thêm thuộc tính !important vào CSS để ép màu chữ hiển thị rõ ràng trên nền tối.
- Sửa lại toàn bộ nội dung text và địa chỉ thực tế của học viện thành CDAcademy.

### 5. Phần nào em tự viết?
- Tự tạo cấu trúc thư mục, đặt tên repository và liên kết các file HTML/CSS/JS lại với nhau.
- Tự nhúng thư viện Bootstrap CDN và test thủ công giao diện responsive trên điện thoại.

### 6. Em học được gì?
- Biết cách dùng hệ thống lưới (Grid) của Bootstrap để làm responsive.
- Hiểu cơ chế lọc mảng trong JS và cách lưu/xóa dữ liệu bằng LocalStorage.
- Biết cách xử lý độ tương phản màu sắc khi làm giao diện tối (Dark Mode).
